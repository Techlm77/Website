Database

This assignment required me to create a database for a Home Group. The Home Group needs to be able to record and access all the details of the properties from their different development.


I started by creating an ERD (Entity Relationship Diagram) to design a database then I created a wireframe for the data entry forms. I continued by designing a main menu enabling the user to add, remove and view data from the database. The database was fully tested and proved to work well.
